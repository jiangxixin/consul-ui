Prometheus 配置管理
简介 

简单的使用consul api 对 service进行增删改

prometheus 启动参数增加:

--web.enable-lifecycle

prometheus 的配置文件如下

scrape_configs:
  - job_name: 'prometheus'
    static_configs:
    - targets: ['localhost:9090']

  - job_name: 'consul-services'
    consul_sd_configs:
    - server:   '192.168.0.200:8500'
      services: []
    relabel_configs:
    - source_labels: [__meta_consul_tags]
      regex: .*test.*
      action: keep
